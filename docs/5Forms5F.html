5.F 3.3.1-error-identification

Test Name Test ID Test Condition
3.3.1-error-identification 5.F The item in error is identified in text and sufficiently described to the user in text.

Evaluate Results:

If the following is TRUE, the content PASSES:

  1. The item that is in error is identified in text and sufficiently described to the user in text.

Tips:

  • Remember-the error message may need to be tested as part of 5.E, 4.1.2-change-notify-forms.
  • If the web page does not have automatic error detection, this is marked DOES NOT APPLY.
  • Providing only a symbol (such as ! or X) or red outline for the form field is error is not sufficient; it must be accompanied with text describing the error (such as: X Incorrect username).
  • This test is NOT verifying that the error message also suggests how to fix the error.
  • It may help to understand the disabilities this Success Criteria benefits:
    • Provides information about input errors in text which allows users who are blind or colorblind to perceive the fact that an error occurred.
    • May help people with cognitive, language, and learning disabilities who have difficulty understanding the meaning represented by icons and other visual cues.
